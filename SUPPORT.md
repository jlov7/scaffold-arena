# Support

## Documentation First

Before requesting support, check:
- [`README.md`](README.md)
- [`docs/README.md`](docs/README.md)
- [`docs/getting-started.md`](docs/getting-started.md)
- [`docs/user-guide.md`](docs/user-guide.md)

## How to Ask for Help

When opening a support request, include:
- Environment (OS, Node, Python, package manager versions)
- Exact command run
- Full error output
- Reproduction steps

## Categories

- Usage questions: how to run or interpret outputs
- Engineering questions: architecture, extension points, testing
- Reliability questions: SSE, retries, performance budgets
- Security questions: use private channel per [`SECURITY.md`](SECURITY.md)

## Not Included

- 24/7 operational support
- SLA-backed response guarantees
